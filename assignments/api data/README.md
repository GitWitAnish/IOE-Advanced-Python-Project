# Football API Data Fetch

This folder contains a script that fetches UEFA Champions League match data from the ESPN public API and saves it as CSV.

## Files

- football.py: Fetches data from API and writes CSV.
- champions_league_matches.csv: Generated output file.

## Data Source

- Endpoint: https://site.api.espn.com/apis/site/v2/sports/soccer/uefa.champions/scoreboard


## Output Columns

- date
- home_team
- away_team
- home_goals
- away_goals
- stadium
